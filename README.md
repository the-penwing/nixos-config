# nixos-config

Personal NixOS configuration for `nixos-t14s`* — a Wayland-only Hyprland + Caelestia setup managed with flakes + home-manager, dotfiles synced via `scripts/sync-dotfiles.sh`, and micro:bit-focused dev shells alongside a curated ethical-hacking lab toolset in `modules/system/packages.nix`.
